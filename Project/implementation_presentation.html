Implementation Details

Embedding Generation

Index Building (build_index_v2.py)

# Initialize embedder embedder = SentenceTransformer("all-MiniLM-L6-v2") # Embed all text chunks def embed_texts(texts): return embedder.encode( texts, convert_to_numpy=True ).astype("float32") # Create embeddings for all chunks embeddings = embed_texts([t[3] for t in texts]) # Shape: (n_chunks, 384) # Build FAISS index dim = embeddings.shape[1] # 384 index = faiss.IndexFlatL2(dim) index.add(embeddings)

Query Search (mcp_server_v2.py)

# Embed query query_emb = embedder.encode([query]).astype("float32") # Search for similar vectors distances, indices = index.search(query_emb, k=10)

Key Points:

  • Same model used for indexing and querying
  • 384-dimensional vectors
  • L2 distance for similarity
  • Fast search (<100ms for 1,200+ vectors)
